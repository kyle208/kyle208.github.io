高级娱乐-智能聊天

1、基础设置​

开启/关闭聊天
开启/关闭随机聊天
开启/关闭艾特聊天
开启/关闭私聊聊天
切换青云客/茉莉云/小爱/chatgpt

小爱模式切换：
切换小爱语音/文本

首次使用可点击【★官方预设】 按照官方预设更换你自己喜欢的

艾特聊天可先艾特或后艾特机器人，在输入文本，都可回复

2、青云客设置​

此为对接官方，只需设置机器人昵称及性别

3、茉莉设置​

此为对接官方，点击按钮前往注册，将2个码填写保存即可

4、小爱设置​

此为对接api，部分服务器无法连接，请自行测试

不支持的框架，无法发送语音

5、ChatGPT设置​

此为对接api及官方，官方可能需要翻墙等操作才可使用，自行测试

6、文心一言设置​

请自行前往官方注册获取key，填入即可使用

此手册适用于：Ver:2.4.1

简介​

这是一款支持多框架的插件。支持多个QQ框架、VLW微信系列框架、鲲鹏微信系列框架，只需更改尾缀即可使用

说明​

此插件默认文件名是Myqq框架的，你可以更改尾缀来适配到其他PC框架，尾缀请看下方

适配QQ框架

myqq[二开版]/ono/ER/黎明DawnBot/VLQ/smallqq/starqq/小栗子【推荐mq/小栗子框架】

适配微信框架

vlw/鲲鹏/千寻/可爱猫/mywechat

适配微信框架

【只建议前两个框架，其他框架功能缺失，无法加入，但大部分功能正常使用，推荐vlw格式，目前为西瓜】

可在此处下载对应框架： 下载地址

1.请将文件放到：（通常情况下）

xx框架的Plugin目录下，鲲鹏为app目录

格式为：kyle.框架缩写.dll,大小写无所谓哦

序号示例插件名适配框架类型备注
1kyle.xg.dll西瓜微信推荐
2kyle.kp.dll鲲鹏微信推荐
3kyle.mw.dllmywechat微信-
4kyle.zy.dll智衍微信-
5kyle.qx.dll千寻微信-
6kyle.xlz.dll小栗子单Q安卓QQ-
7kyle.lb.dll蜡笔小新安卓QQ推荐
8kyle.qy.dllQY框架安卓QQ-
9kyle.dlloaz安卓QQ-
10kyle.mq.dllMyqqPCQQ-
11kyle.wq.dllWolfBotPCQQ已无
12kyle.sa.dllsmallqqPCQQ-
13kyle.star.dllstarqqPCQQ-
14kyle.opc.dllOPCPCQQ-
稳定性及功能

微信：vlw(西瓜)≈kp > mw>千寻

QQ：mq=小栗子 > 其他

最简易辨别是否兼容方法：兼容mq，小栗子即可使用本插件，很多框架都是兼容的，或小栗子改的

反馈​

请将以下内容都发送给作者：

bug反馈步骤：
1.使用的框架【判断部分功能是否生效】
2.发送的功能【帮助作者判断具体哪段代码问题】
3.机器人的反应【帮助作者判断具体哪段代码问题】
4.日志及设置截图【获取你是官方配置还是自定义导致的问题】

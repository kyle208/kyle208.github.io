首页

首次启用框架，并打开设置，让其创建工作目录

然后按照下方步骤开始使用！

1、打开窗口后，直接私聊机器人发送：【设置主人】来设置

2、根据【设置主人】后的提示继续操作

3、默认界面为全群启用+不开启艾特回复+限制回复

4、在【2启用以下群】及【3不启用以下群】时，需在机器人所在群发送【开启群】开启不同模式

5、出租模式，123模式将不可用，请私聊机器人发送【出租系统】查看。此指令在菜单中，如不显示，请加载最新菜单

5.1、出租使用名单：在开启出租模式后可用

1、首页设置​

提示

改变下拉【选中群】更改分群及全局显示

模式1：适用于全部群使用

模式2：适用于只开启某几个群下使用

模式3：适用于不开启某几个群，其余群使用

精准开关群模式：适用于模式2/3下，再去精准控制单群的功能【不会不懂的千万别勾选】

模式4：仅适用于出租，以上模式将失效，开启菜单后私聊机器人【出租系统】查看

出租使用名单：在开启出租模式后可用，并开启开关

2、白名单​

1、界面设置你要加的白名单用户qq/微信id或指令添加

勾选后，除了个别极度敏感的指令外，管理员基本可控制机器人功能

1、界面设置机器人管理员或指令添加

指令

添加/删除管理员@10086

4、出租使用名单​

1、界面设置你要加的出租使用名单qq/微信id或指令添加

指令

5、全局表情获取提醒​

提示

QQ/微信框架下表情获取：可丰富如菜单等可自定义回复的文本

在群内或私聊机器人，发送一个表情后，看日志输出的结果，将结果复制到你要添加的位置即可
